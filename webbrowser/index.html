Web Browser CS

Have you ever wanted to use both modern Chromium AND old Internet Explorer in the same program?

Well, search no more, we have the solution for you: WebBrowserCS

Navigation:

Features

IE Tab: For your Legacy browser needs, IE Tab can use any IE version from 7 up (by default set to 7)

Chromium tab: in case you ever need to use any html5 pages

A basic text editor: beware that it is kind of slow in loading files...

Specifications

IE mode: IE7 to IE11
Chromium mode: Chromium 129 (CEFSharp 129)

Requirements

Microsoft .net runtime: 4.7
Platform: 32 or 64 bit, NO ARM (yet)
